Ever had a friend or relative who misspells your name, forgotten how to spell playwright, or confused Timothy Olyphant with Timothy Elephant? Let's write a program that helps us correct these misspelled words. You'll be given a word bank of names and a misspelled name. Use your coding skills to find the correctly spelled name.

You are given a word box of names and a possibly misspelled name. Find the closest name if any.

Input: A word box of possible names separated by spaces (unicode). The misspelled name as a string (unicode, without spaces).

Output: The closest name if any is close, otherwise the empty string "".

Example: 

checkio("Arthur Lancelot Robin Bedevere Galahad","Lance-a-lot") == "Lancelot"
checkio("Arthur Lancelot Robin Bedevere Galahad","Lionel") == ""

How it is used: This is a simple exercise in working with strings and using the Python standard library to solve common problems without considering the details. This could be used in trivia programs to determine the intended user response while allowing for misspellings.

Precondition: 0 < len(name) ≤ 1000 0 < len(word_box) ≤ 1000
all(ch in string.ascii_letters+'-' for ch in name) all(ch in string.ascii_letters+'- ' for ch in word_box)
